# 08 - Prompt library for static app

```text
Generate a prompt library for GOELRAH optimized for a static frontend AI coding assistant.

Create prompt groups for:
- general coding help
- repo explanation
- bug diagnosis
- refactoring
- test generation
- documentation writing
- commit message generation
- code review
- safe coding / secure coding
- extension-related prompts for future Phase 2

For each prompt include:
- name
- purpose
- recommended inputs
- output structure
- risk notes
- example use

Keep prompts practical and production-oriented.
```
